Method 1: Browser DevTools (Recommended)

  1. Open Chrome DevTools (F12 or Cmd+Option+I)
  2. Click the device toolbar icon (Cmd+Shift+M)
  3. Set dimensions to 1280x800 or 640x400
  4. Use browser's screenshot feature or external tool

Method 2: macOS Screenshot

  1. Press Cmd+Shift+4
  2. Press Space to capture window
  3. Click on the browser window
  4. Crop to exact dimensions in Preview

Method 3: Screenshot Extension

  1. Install a screenshot extension (e.g., Awesome Screenshot)
  2. Capture visible area or full page
  3. Crop to required dimensions

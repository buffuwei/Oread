## Detailed Description

Transform your web reading into organized knowledge with Obsidian Read Later - the intelligent web clipper that saves articles directly to your Obsidian vault with AI-generated summaries.

### üéØ Key Features

**Smart Content Extraction**
- Automatically extracts article content using Mozilla's Readability algorithm
- Filters out ads, navigation, and clutter
- Intelligently selects important images (filters decorative elements)
- Preserves article structure and formatting

**AI-Powered Summaries**
- Generate concise summaries of any article with one click
- Supports multiple LLM providers:
  - OpenAI (GPT-3.5, GPT-4)
  - Anthropic Claude
  - Local LLMs (Ollama, LM Studio, vLLM, LocalAI)
  - Any OpenAI-compatible API endpoint
- Privacy-focused: Use local models to keep your data private

**Perfect Obsidian Integration**
- Generates properly formatted Markdown files
- Includes YAML frontmatter with metadata (title, URL, author, date, tags)
- Saves directly to your configured vault folder
- Compatible with all Obsidian themes and plugins

**Preview & Edit Before Saving**
- Optional preview mode lets you review content before saving
- Edit AI summaries and content in a side panel
- See exactly what will be saved to your vault
- Toggle between rendered preview and source editing

**Flexible Configuration**
- Easy setup with intuitive configuration interface
- Support for custom API endpoints (perfect for self-hosted solutions)
- Optional API key (local LLMs don't need authentication)
- Customizable save folder and file naming

## Privacy Policy

### Data Collection and Usage

Obsidian Read Later does NOT collect, store, or transmit any user data to external servers operated by the extension developer.

**What We Access:**
- Content from web pages you choose to save (processed locally)
- Your configuration settings (stored locally in Chrome)

**What We Do NOT Do:**
- We do not collect personal information
- We do not track your browsing history
- We do not send data to our servers (we don't operate any servers)
- We do not sell or share your data with third parties

**Third-Party Services:**
When you use AI summary features, content is sent to your chosen LLM provider:
- OpenAI (if you select OpenAI as provider)
- Anthropic (if you select Claude as provider)
- Your custom endpoint (if you configure a custom API)

These services have their own privacy policies. We recommend reviewing them:
- OpenAI Privacy Policy: https://openai.com/privacy
- Anthropic Privacy Policy: https://www.anthropic.com/privacy

**Local LLM Option:**
For maximum privacy, you can use local LLMs (like Ollama) which process everything on your own computer without sending data to external services.

**Data Storage:**
- API keys and configuration are stored using Chrome's encrypted storage API
- No data is stored on external servers
- All processing happens locally in your browser

**Open Source:**
This extension is open source. You can review the code to verify our privacy claims.
